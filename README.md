## 🚀 ویژگی‌ها

- 📄 تبدیل صفحات PDF به تصویر  
- 🔍 استخراج متن فارسی از تصویر با Tesseract OCR  
- ✨ خلاصه‌سازی دقیق محتوا با استفاده از GPT-3.5 یا GPT-4  
- 🧪 ساختار ماژولار و قابل گسترش  
- 💻 اجرای آسان در محیط لوکال  

---

## 🛠 پیش‌نیازها

- Python 3.10+
